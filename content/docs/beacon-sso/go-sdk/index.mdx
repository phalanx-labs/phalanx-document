# Go SDK

本页只讲一件事: 你的 Gin 服务如何接入并使用 Beacon SSO Go SDK。

## 你会用到的能力

| 需求 | 对应能力 | 文档 |
|------|------|------|
| 快速完成接入 | 安装、环境变量、启动示例 | [快速开始](./quick-start) |
| 让接口必须登录后访问 | `middleware.CheckAuth(ctx)` | [认证检查中间件](./middleware/auth-check) |
| 打通登录/回调/登出 | `Route.OAuthRouter()` | [OAuth2 路由](./oauth2/routes) |
| 调用用户/账户/商户接口 | `Route.UserRouter()` / `Route.AccountRouter()` / gRPC 客户端 | [gRPC 服务](./grpc) |
| 配置环境参数 | `SSO_*` 环境变量 | [环境变量配置](./environment) |

## 接入步骤概览

1. 引入依赖: `go get github.com/phalanx-labs/beacon-sso-sdk`
2. 在启动节点中追加 `bSdkStartup.NewStartupConfig()...`
3. 用 `bSdkRoute.NewRoute(ctx)` 挂载 SDK 路由
4. 对受保护业务路由加 `CheckAuth` 中间件

## 快速开始

<Callout type="info">
先按 [快速开始](./quick-start) 走完整接入，然后按业务需要看 `OAuth2`、`gRPC`、`中间件` 子文档。
</Callout>
