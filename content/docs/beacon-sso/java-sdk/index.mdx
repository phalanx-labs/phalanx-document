# Java SDK

本页只回答“接入后怎么用”。

## 你会用到的能力

| 需求 | 对应能力 | 文档 |
|------|------|------|
| 5 分钟跑通接入 | Starter 依赖 + `application.yml` | [快速开始](./quick-start) |
| 配置端点/路径/gRPC | `beacon.sso.*` | [配置参考](./configuration) |
| 统一鉴权入口 | `BeaconSsoFilter` | [认证过滤器](./filter/auth-filter) |
| Controller 里直接拿用户信息 | `@InjectData` | [@InjectData](./aop/inject-data) |
| Controller 做权限校验 | `@PermissionVerify` | [@PermissionVerify](./aop/permission-verify) |

## 接入路径概览

1. 引入 `bamboo-sso-sdk-springboot`
2. 配置 `beacon.sso.*`
3. 启动应用，自动注册 Filter、Controller、AOP
4. 在业务 Controller 使用 `@InjectData` 和 `@PermissionVerify`

## 版本信息

- 当前版本: 0.0.1-SNAPSHOT
- Java 版本: 17+
- Spring Boot: 3.5.11
- Maven 坐标: `com.frontleaves.phalanx.beacon.sso:bamboo-sso-sdk-springboot`
